Hotel Trends

This 3-month data analysis compares current hotel bookings against historical performance to uncover trends in room mix, pricing, and demand.

  • "On The Book" Mix vs Last Year: Segment performance vs same period LY
  • Current Mix vs LY Forecast & Actual: Benchmark against LY expectations
  • OTB Variances vs 1 Period Ago: Week-over-week changes
  • OTB Mix %: Proportion of each segment (OTA, Corporate, etc.)
  • Segment ADR + Change from Last Week: Trends and pricing movement

Tools Used: SQL (PostgreSQL), Python (Pandas, Seaborn), Tableau 2019, HTML, CSS, Bootstrap.
